## Commit Message Rules

1. **Language Requirement**: All commit messages MUST be written in Chinese
2. **Format Guidelines**:
   - Start with concise verbs (e.g., 添加, 修复, 更新, 删除, 优化, 重构)
   - Keep the subject line under 50 characters
   - Leave a blank line between subject and body
   - Use Chinese punctuation in the body
3. **Type Prefixes** (optional):
   - feat: 添加新功能
   - fix: 修复问题
   - docs: 文档变更
   - style: 代码格式（不影响功能）
   - refactor: 重构
   - perf: 性能优化
   - test: 测试相关
   - chore: 构建/工具变更
